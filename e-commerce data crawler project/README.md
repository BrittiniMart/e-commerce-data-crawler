# E-commerce Data Crawler System

## Project Overview
A complete Python web crawler system for collecting and analyzing e-commerce product data from multiple platforms.

## Features
- Multi-platform crawling (Amazon, JD.com simulation)
- Anti-crawling protection (User-agent rotation, request delays)
- SQLite database storage
- Price distribution analysis
- Basic sentiment analysis
- Flask web dashboard

## Installation
```bash
git clone https://github.com/yourusername/ecommerce-data-crawler.git
cd ecommerce-data-crawler
